# [15] 테스트 모니터링 / 제어 및 테스트 종료

### 1. 테스트 모니터링 및 제어

✏️ *계획 대비 차이가 있는 경우 **계획된 테스트 활동을 조정**하는 것이 적합 (한 사이클 동안 유지x !!)*

### 1-1.  테스트 모니터링

테스트 계획에서 테스트활동의 실제 상황을 객관적으로 분석하고 판단하기 위해 **테스트 메트릭** 정의했음 → **정량적인 모니터링** 가능

### 1-2. 테스트 활동 제어

테스트 활동 제어 유형

- 동적 테스트 활동의 **시작**
- 동적 테스트 활동의 **종료**
- 테스트 **계획과의 차이**
- **위험 관리**

테스트 활동 제어 방법

- **테스트 컨텍스트 관점**의 테스트 활동 제어
- **테스트 전략 관점**의 테스트 활동 제어
- **테스트 수행 계획**의 테스트 활동 제어

### 1-3. 테스트 현황 보고

- 보고 대상 **기간**
- 계획 대비 **진척도**
- **테스트 메트릭**
- 신규 및 변경 위험
- ✏️ *이후 테스트 계획*

### 2. 테스트 종료

결과물을 이후 프로젝트에서도 재사용할 수 있도록 테스트 자산으로 관리

### 2-1. 테스트 자산 관리

테스트 자산을 **형상 관리 대상 항목**으로 식별하고 형상 관리 방법 적용 → **고유의 식별자 부여** → 공유 시스템에 **저장** → 적절한 형상 통제 수행, 형상 상태 보고
