## 3) 테스트 전략 수집
테스트 전략 항목
### 1️⃣ 개별 테스트
### 2️⃣ 테스트 산출물
### 3️⃣ 테스트 설계 기법
정적 테스트 설계 기법과 동적 테스트 설계 기법으로 분류됨.

테스트 설계 기법 자체의 특성도 고려할 필요 있음
- 상태 의존적 동작 여부
- 동작 방식
- 테스트 변수 간의 제약사항 고려 여부
### 4️⃣ 테스트 환경 요건
시스템 테스트와 인수 테스트는 비기능적인 요구사항에 대한 테스트도 수행됨
### 5️⃣ 테스트 데이터 요건
- 테스트 대상 입력
- 테스트 대상 상태
- 테스트 환경 항목 상태
### 6️⃣ 재테스팅 및 리그레션 테스팅
  #### 재테스팅 (Re-testing)
  결함을 검출한 테스트 절차(테스트 케이스)를 활용하여 다시 테스팅을 수행해야. 테스트 절차를 자동화 도구를 통해서 실행시킬 수 있도록 **테스트 스크립트(Test Script)**를 개발하여 실행

  #### 리그레션 테스팅 (Regression Testing)
  전체 시스템이 여전히 정상적으로 작동하는지 확인, **소프트웨어 변경이 발생한 후**에 의도하지않은 결함 안만들면서 + 기존의 요구사항 만족시키는지

  ✏️ *개별 테스트에 대한 계획서보다는 프로젝트 테스트 계획서에 기술함*
### 7️⃣ 테스팅 중단 및 재시작 조건
ex) 과도한 비율로 테스트가 실패할 때 해당 테스트 중단 → 결함 해결 후 테스트 전체 재시작

ex) 기존 시나리오 테스트 실패 시 대안 시나리오 테스트 중단 → 결함 해결 후 해당 유스케이스의 모든 시나리오 테스트 재시작

ex) 테스트 환경 문제로 실행 시 동일 테스트 환경 사용하는 모든 테스트 중단 → 환경 문제 해결 후 동일 테스트 환경 사용하는 모든 테스트 재시작

### 8️⃣ 테스트 메트릭
테스트 계획, 테스트 설계 및 구현, 테스트 환경 구축 및 관리, 테스트 실행 그리고 결함 관리보고 활동 별로 매트릭 결정

진척도 파악 good, 테스트 완료 여부에 대한 평가 데이터 활용 good
  #### 테스트 계획 활동 메트릭
  - 테스트 대상 수
  - 피처 수

  #### 테스트 설계 및 구현 활동 메트릭
  피처 집합 수, 테스트 케이스 수, 테스트 절차 수, 테스트 환경 항목 수, 테스트 데이터 수

  테스트 케이스 및 테스트 절차의 적합성, 테스트 케이스+절차가 목표로 하는 범위를 충분히 반영하였는지 **커버리지(Coverage)** 통하여 측정 가능

  ❶ **요구사항 커버리지**

  - **유스케이스** 커버리지
  - **시나리오** 커버리지
  - **비즈니스 규칙** 커버리지

  ❷ **설계 커버리지**
  - 설계 단위 **수행** 커버리지
  = (1회 이상 확인된 함수의 수)/(전체 함수의 수)
  - 설계 단위 **호출** 커버리지
  = (1회 이상 호출된 함수의 수)/(전체 함수 호출의 수)

  ❸ **코드 커버리지**
  - **문장** 커버리지
  - **결정** 커버리지
  - **MCDC**

  #### 테스트 환경 구축 및 관리 활동 메트릭
  - 테스트 환경 구축률
  - 테스트 데이터 준비율

  #### 테스트 실행 활동 메트릭

  #### 결함 보고 활동 메트릭
  **누적 검출 결함 수** : 전체 기간에 걸쳐서 식별된 모든 결함의 수
  - 검출 결함 수
  - 검출 결함 밀도 : (검출된 결함 개수)/(대상 코드 행수_KLOC)
  - 상태별 결함 수
  - **결함 나이** : ✏️*결함이 보고되고 종결될때까지 걸리는 시간(opening time~closing time)*


### 9️⃣ 테스트 완료 기준

  #### 테스트 케이스(테스트 절차) 기반 방법
  #### 테스트 커버리지 기반 방법 
  #### 결함 기반 방법


- 분석 유형
  #### 신뢰도 예측 모델 기반 방법
  #### 결함 탐침 기반 방법
  #### 복수 테스트팀 기반 방법
  ```
  Pa = Nab / Nb
  Pb = Nab / Na
  시스템 존재 전체 결함 수 N = (Na * Nb) / Nab
  발견하지 못한 결함의 수 N(a)(b) = (Na - Nab) * (Nb - Nab) / Nab
  결함 발견하지 못할 비율 P(a)(b) = (Na - Nab) * (Nb - Nab) / (Na * Nb)
  ```
