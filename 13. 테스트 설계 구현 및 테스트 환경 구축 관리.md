# [13] 테스트 설계/구현 및 테스트 환경 구축/관리

### 피처

테스트 대상에 대한 테스트 범위

✏️ 오답노트

- *세부 피처의 내용으로 **무엇을 테스트할지** 기술*
- *세부 피처의 **우선순위** 기술*
- *내용에 대한 **테스트 베이시스의** 해당 항목에 대한 **참조** 기술*

### 피처 집합

- 유사한 성격 가진 복수 개의 피처 묶어서 정의, **식별된 피처들의 논리적 묶음**
- **독립적인 테스트를 수행하는 단위**
- 하나의 피처 집합은 세부 피처를 확인하기 위한 테스트 절차로 구성

✏️ ***테스트 설계 기법**은 **피처 집합**별로 기술 !!!!*

✏️ *동일한 테스트 재시작 조건이 적용되는 피처들은 동일한 피처 집합에 포함*

테스트 계획의 각 피처 구체화 → 복수 개의 세부 피처 식별 

**각 세부 피처 → 복수 개의 테스트 케이**스 개발

### 테스트 절차

동일한 피처 집합에 속한 **테스트 케이스들의 시퀀스, 테스트 실행 단위**

---

### 1. 테스트 설계

✏️ *코딩 스타일의 예로 올바르지 않은 것은 널 역참조 ..*

### 1-1) 테스트 설계

테스트 **범위** + 테스트 **전략 구체화**

### 1️⃣ 피처 집합 식별

- 이후의 테스트 활동 수행할 때 **독립적으로 간주**
- 테스트 설계 및 구현, 테스트 환경 구축 및 관리, 테스트 실행 활동의 단위
- **상호의존적으로 확인이 필요한 피처들은 동일한 피처 집합에** (다른 피처 집합에 속한 피처들은 독립적으로 테스트)
- **동일 테스트 전략을 적용하는 피처들은 동일한 피처 집합에** (상이한 테스트 전략 적용해야하는 피처들은 다른 피처집합에 포함)
- 피처 집합 식별한 후에는 각 **피처 집합별로 우선순위 부여** (위험 분석 결과를 바탕으로 결정)

### 2️⃣ 피처 구체화

각 피처별로 세부 피처 정의, ✏️*세부 피처에 대한 우선순위*도 위험 분석 결과 바탕으로 결정

### 3️⃣ 테스트 전략 구체화

피처 집합에 포함된 피처 특성 고려하여 구체화 + 테스트 설계 기법 구체화

**테스트 설계 명세서** *→ ✏️ **피처 집합**별로 설계 결과 기술*

설계 작업의 결과물 기록. 목적 + 우선순위 + 추적성 + 구체적 테스트 전략 + 피처 목록

✏️ ***테스트 데이터 준비는 테스트 실행 단계 !!***

✏️ *테스트 수행 시 사용될 **테스트 케이스에 대한 생성 및 확보 전략** 수립*

✏️ ***테스트 목적 달성하기 위한 구체적 테스트 접근방법** 정의*

✏️ *테스트 수행하기 위한 **테스트 절차서** 작성*

### 1-2) 테스트 케이스 개발

테스트 대상에 존재하는 결함 검출 + 품질 확인하기 위한 게 목적

테스트 케이스 항목

- **목적** (테스트하고자 하는 구체적 상황)
- **추적성** (테스트 베이시스의 해당 항목에 대한 참조 나열)
- **우선순위** (우선순위가 높은 TC부터 실행)
- **선행 조건** (테스트 케이스가 실행되기 위해서 테스트 대상 및 테스트 환경 항목에 요구되는 특정상태 기술)
- **입력** (입력하는 행위도 정의 !)
- **예상 결과** (허용 오차가 있다면 해당 오차도 기술, 결과 비교하기 위해서 필요한 행위도 기술)

### 1-3) 테스트 절차 개발

테스트 절차 항목

- **목적**
- **우선순위**
- **시작 작업**
- **테스트 케이스 목록** (개별적인 선행 조건 정의)
- **종료 및 정리 작업**
- **다른 테스트 절차와의 관계** (의존관계)

### 1-4) 테스트 환경 요건 명세

- 테스트 환경 :  HW + SW
- 외부 연동 시스템
- **공존 소프트웨어**
- **테스트 도구**

**테스트 환경 요건 명세서**

테스트 환경 항목명 + 설명 + 요구사항 + 필요 시기 + 담당자

### 1-5) 테스트 데이터 요건 명세 작업

테스트 데이터 요건

- 테스트 **대상에 대한 입력** 데이터
- 테스트 **대상의 상태** 데이터
- 테스트 **환경 항목의 상태** 데이터

**테스트 데이터 요건 명세서**

테스트 데이터명 + 설명 + 요구사항 + 초기화 필요 여부 + 보관 필요 여부 + 담당자
